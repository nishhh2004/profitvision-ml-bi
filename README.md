## 📌 Project Overview
**ProfitVision** is an end-to-end analytics project that bridges predictive modeling with executive-level visualization. This project analyzes startup financial data to identify which operational expenses—R&D, Administration, or Marketing—most significantly drive profitability.

The project is structured into two core components:
1. **Predictive Analysis (Python):** Building a Multiple Linear Regression model to identify statistical correlations and predict profit.
2. **Executive Visualization (Power BI):** Designing a premium, interactive dashboard to communicate these findings for business decision-making.

## 🔬 ML Methodology
The analysis was performed using Python to ensure accuracy and statistical rigor.

**Technical Stack:** `pandas`, `NumPy`, `scikit-learn`, `matplotlib`

**Process:**
1. **Data Preprocessing:** Handled categorical variables and normalized financial inputs.
2. **Model Training:** Utilized a Multiple Linear Regression model to establish feature importance.
3. **Insights:** Identified that R&D spending serves as the primary driver for profitability, while high marketing spend shows diminishing returns.
